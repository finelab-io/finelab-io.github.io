Product API v1.0.0

Scroll down for code samples, example requests and responses. Select a language for code samples from the tabs above or the mobile navigation menu.

The Product API exposes catalog, products and prices information.

Base URLs:

  • https://test-api.finelab.io

  • https://api.finelab.io

Email: API support License: Apache 2.0

Authentication

  • oAuth2 authentication.

    • Flow: clientCredentials

    • Token URL = https://finelab-test.eu.auth0.com/oauth/token

ScopeScope Description
read:catalogAllows to read catalog
read:productAllows to read product
read:priceAllows to read price
