Catalog API v1.0.0

Posted Sep 4, 2022 Updated Sep 11, 2022
By API
5 min read

The Catalog API allows you recieve products from the finelab product catalog. Base URLs:

  • http://localhost:8080

  • https://test-api.finelab.io

Email: API support License: Apache 2.0

Authentication

  • oAuth2 authentication.

    • Flow: clientCredentials

    • Token URL = https://finelab-test.eu.auth0.com/oauth/token

ScopeScope Description
read:catalogAllows to read catalog
